# Dramaturgia para la Danza — Taller Intensivo

Página del taller "Dramaturgia para la Danza" de RAÍZ Colectiva Teatro. Sitio estático (HTML/CSS/JS sin build), listo para desplegar en Netlify conectando este repositorio.

## Desarrollo local

Abre `index.html` directamente en el navegador, o sirve la carpeta con cualquier servidor estático, por ejemplo:

```
python -m http.server 8080
```

## Despliegue en Netlify

1. Sube esta carpeta a un repositorio de GitHub.
2. En Netlify: "Add new site" → "Import an existing project" → selecciona el repositorio.
3. Build command: (vacío). Publish directory: `.` (raíz del repo).
